REDOT

Redot stands for Reactive Document Templates. It is a javascript API (Application Interface) and an embedded DSL (domain specific language) that introduces reactive capabilities to document generation driven by data. The arrangement of generated DOM fragments reflects the data structure. Modifications to underlying data automatically change view layer. Redot automatically manages the dependency between data object expressions and DOM fragments.
